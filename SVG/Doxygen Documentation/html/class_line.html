Detailed Description

Клас Line. Класът Line наследява класа Shape. Той съдържа допълнителни променливи, като координати на две точки, които при свързването им образуват Line.

Конструктор за класа Line. Конструкторът приема допълнителни променливите необходими за съществуването на една линия в равнината, като координати на две точки, които при свързването им образуват Line. Line може да има подчертаваща линия, която е с цвят и широчина на щриха. Тези компоненти са наследени от класа Shape.
