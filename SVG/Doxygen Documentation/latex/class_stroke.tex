\doxysubsection{Detailed Description}
Клас \mbox{\hyperlink{class_stroke}{Stroke}}. Класът \mbox{\hyperlink{class_stroke}{Stroke}} представлява подчертаващата линия на всеки обект(\+Shape). \mbox{\hyperlink{class_stroke}{Stroke}} може да има цвят и широчина на щриха.
