Detailed Description

Parameters
xThe x property describes the horizontal coordinate of the position of the element.
yThe y property describes the vertical coordinate of the position of the element.
widthThis property specifies the width of the rectangle. Negative values for 'width' are illegal.
heightThis property specifies the height of the rectangle. Negative values for 'height' are illegal.
rxThe rx property describes the horizontal curve radius of the rectangle. A negative value for rx is illegal.
ryThe ry property describes the vertical curve radius of the rectangle. A negative value for ry is illegal.
